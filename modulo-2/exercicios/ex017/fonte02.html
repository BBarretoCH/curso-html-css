Fontes Externas

Site útil: Dafont

É necessário verificar sempre se as fontes possuem glifos com acentuação

  • Baixar no site de interesse a fonte desejada
  • descompactar e colocar o arquivo da fonte dentro da pasta do projeto
  • Dentro do VS Code colocar a seguinte regra
    • @font-face{
      font-family: nome desejado;
      src: URL(path da fonte.format) format (otf ou ttf) format = opentype(otf), truetype(truetype)
      font-weight: normal;
      font-style: normal;
      }
