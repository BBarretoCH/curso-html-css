RESUMO

Nessa aula nós trabalhamos a CSS interna, a qual é aplicada na área da tag HEAD do nosso código com o uso da tag <style>. A vantagem dessa estratégia é que ao criarmos um estilo para um seletor da tag de interesse (body { }) aplicaremos as mudanças para tudo que estiver contido na tag BODY. Sempre que desejarmos aplicar estilo para uma tag podemos criar o seletor dessa tag. Mas essa estratégia possui desvantagens. Ao utilizarmos a CSS interna ou local se caso tivermos mais de uma página, devemos copiar a tag style e todo o seu contéudo para todas as páginas que quisermos aplicar. E novamente, em caso de necessidade de alteração de estilos de uma determinada página, deverá ser feito em cada página de interesse.

Extra

Aqui irei colocar alguns parâmetros utilizados para mudar a aparência deste conteúdo

  • background-color = muda a cor de fundo
  • font-size = tamanho da fonte
  • font-family = tipo de fonte
  • letter-spacing = espaçamento entre as letras
  • magin-top = margem do cima do parágrafo
  • margin-bottom = margem de baixo do parágrafo
  • text-indent = Espaçamento do inicio do parágrafo
  • text-align = alinhamento do texto
  • line-height = espaçamento entre as linhas
  • color = determina a cor
