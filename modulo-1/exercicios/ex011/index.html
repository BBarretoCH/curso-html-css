Imagem Dinâmica

Tente abrir esse site em vários dispositivos diferentes ou simplesmente aumente e diminua o tamanho do seu navegador

RESUMO

Nesse exercício foram utilizadas imagens de tamanhos diferentes para adapta-las ao tamanho da tela do usuário. Conforme era detectado o tamanho da tela de exibição a imagem se adequava automaticamente. Para tal foi utilizada a tag <picture>. Primeiramente, dentro da tag <picture> foi inserida a tag img (já vista anteriormente) e para esse exercício, foi selecionada a maior imagem disponível. Em seguida foi colocada imediatamente acima a tag <source>(Escolhida a source:media:type) e colocada a imagem imediatamente menor. O mesmo procedimento deve se repetir para o número de imagens que deseja adequar o tamanho da tela. Dentro da tag <source> temos o parametro media = max(por padrão vem min)-width: 00px; srcset="imagens/foto-g.png(path da imagem)" e type ="(image/png)tipo de arquivo/extensão".
Os parâmetros escolhidos para esse exercício não são fixos, podem ser outros que irão funcionar do mesmo jeito, somente é necesserário ajustar as necessidades que aparecerão durante a criação do site.

Reproduzindo Audios

Vamos aprender a reproduzir áudios em HTML5

Resumo

Nesse tópico foi mostrado um áudio que foi inserido ao site. Inicialmente foi visitado a bibliioteca de mídia do youtube para adquirir um áudio livre para utilização no projeto. Os passos foram os seguintes:

  • entrar na página www.youtube.com
  • clicar no perfil e ir na opção youtube studio
  • Nos ícones à esquerda da tela clicar em biblioteca de áudio
  • colocar os filtros de interesse e baixar o audio

De posse do(s) áudio(s) de interesse é só utilizar a tag <audio>(semelhante a tag IMG). Caso o navegador nao seja compatível com a reprodução do áudio podemos inserir os parâmetros controls para inserir uma caixa com play para iniciar o áudio; Também há a possibilidade de se colocar o parâmetro autoplay para iniciar o áudio automáticamente.

Vale ressaltar que nem sempre o navegador será compatível com a extensão do áudio que está sendo inserida no site, portanto, é importante mostrar para ele(navegador) que é possível carregar outros formatos. Para fazer isso devemos realizar os seguintes passos:

  • inserir a tag <audio>
  • apagar os parâmetros automáticos que são criados
  • criar um conteúdo aninhado dentro da tag audio com o conteúdo (source:src)
  • No parâmetro src="colocar o path do áudio(midia/happy-mistake.mp3)" e no parâmetro type="colocar o texto relativo ao formato do arquivo(audio/mpeg) ***lembre-se de consultar o site do media type"

Assim o site tentará carregar o primeiro arquivo listado dentro do conteúdo aninhado da tag strong><audio>, se caso não encontrar, tentará carregar o próximo e assim por diante. Se nenhum formato for compatível com o navegador, o áudio não será carregado, mas ainda é possível disponibilizar o áudio para download inserindo um link com a tag <a>.
Ainda vale destacar que dentro da tag strong><audio> podemos inserir o parâmetro preload. Esse parâmetro significa que algo que será carregado(ou não, caso escolhida a opção none)com o site. Aqui vão as opções:

  • Auto -> O site será carregado somente quando todo o conteúdo da mídia for carregado. Isso é ruim porque arquivos grandes podem deixar o site lento
  • Metadata -> irá carregar apenas algumas informações essenciais, sem pesar o carregamento do site
  • none -> não irá carregar nenhuma informação, apenas quando o usuário desejar.
