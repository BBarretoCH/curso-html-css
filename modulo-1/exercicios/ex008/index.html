Principais formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag <B>(essa tag é não semântica, ela não tem significado só possui forma)

Nesta Frase, temos um termo em destaque usando a tag <STRONG>(Essa tag é semântica, possui significado e forma)

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <i>(essa tag é não semântica, ela não tem significado só possui forma)

Nesta frase, temos um termo em ênfase usando a tag <EM>(Essa tag é semântica, possui significado e forma)

Texto Marcado

podemos também criar um texto marcado usando a tag <MARK>.
Aqui nesse caso a marcação ficou cinza pois coloquei uma folha de estilo dentro de uma tag individual, portanto, a marcação de estilo criada na área HEAD nao afetou essa marcação. Caso eu queira colocar uma nova marcação ela terá a cor escolhida pela marcação da folha style criada na parte da tag HEAD.

Mudamos acima a cor da marcação com CSS style dentro da Tag MARK e escolhendo a opção background-color: (cor desejada)
