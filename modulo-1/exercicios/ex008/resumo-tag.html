<B>(Não semântica) e <STRONG>(Semântica) = Negrito ou destaque
<I>(Não semântica) e <EM(Semântica)> = itálico ou ênfase
<MARK> = Marcação de texto
<BIG>(obsoleta) = Texto grande
<SMALL> = Texto Pequeno
<DEL> = Texto Excluído
<INS> = Texto Inserido (Parecido com sublinhado. *Outra tag semântica)
<SUP> = Texto Sobrescrito
<SUB> = Texto Subscrito
<CODE> = Texto com fonte monoespaçada(reserva o mesmo espaço para todos os caracteres)
<PRE> = Texto pré-formatado
<Q> = Texto como sendo uma citação
<BLOCKQUOTE> = Bloco de texto que é uma citação
<ABBR> = Abreviações
