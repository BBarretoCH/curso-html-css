RESUMO

Por fim, trabalhamos nessa aula a aplicação de CSS externa. Para chamar uma CSS para o nosso código devemos criar na área da tag HEAD um link:css. Logo após a criação, o hyperlink de referencia (HRef) aparece com o nome style.css(padrão) e para criar o arquivo podemos segurar CTRL e clicar e o arquivo será criado.

Dentro da página de estilo (Style.CSS)

Um comando recomendado utilizar é o @charset "UTF-8". Esse comando indica que a folha de estilo é compatível com os caracteres presentes no UTF-8. Em seguida podemos colocar todas as características que nosso site deverá ter, tais como, cor de fundo, fonte, posição de imagens e links, etc, com o uso dos seletores para as tags de interesse.

A vantagem é que podemos chamar a folha de estilo com apenas uma linha de comando (link:css) inserido na área da tag HEAD, como dito anteriormente, e pronto, todo o estilo criado no arquivo style.css será aplicado à nossa página.
Não menos importante, devemos destacar a ordem de prioridade que de leitura do código (não a ordem de prioridade de uso) das CSS. A Ordem (maior prioridade para menor prioridade) é CSS Inline -> CSS Interna -> CSS Externa.

Também é possível misturar os estilos(inline, interno e externo) caso sejam necessárias mudanças afunilidas, ou seja, cada vez mais específicas. Por exemplo:

  • temos um site com 15 páginas e aplicamos um estilo para todas elas (CSS externa)
  • Dentre 15 páginas do site, queremos mudar o estilo da página 4 (utilizamos a CSS interna).
  • Dentro da página 7, queremos que o segundo H1 tenha uma cor diferente (aplicamos CSS inline)

E assim por diante.

OBS: Qualquer comando inserido com @ no início será uma regra aplicada à CSS.
