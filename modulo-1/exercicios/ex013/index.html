Resumo

Nessa aula foi tratado sobre CSS inline. Esse tipo de estilo é utilizado para fazer mudanças de estilo na mesma linha. Por exemplo: Caso seja de interesse mudar a cor e o tamanho da fonte do título principal (h1), devemos colocar o parâmetro style dentro de cada tag H1 presente no código. Esse tipo de prática não é recomendada pois irá poluir muito o código, além disso, se caso for necessário fazer alguma mudança posterior, o trabalho será muito grande e teremos que mudar cada linha que contiver a mudança necessária, ou seja, se desejarmos mudar a cor e o tamanho da fonte do H1 decidido anteriormente teremos que fazer isso individualmente para cada H1 que seja de interesse fazer a mudança.

CSS inline são recomendadas apenas para pequenas mudanças pontuais e específicas. Por Exemplo: Desejo mudar o tamanho da fonte de apenas um parágrafo do meu site, então podemos utilizar o style dentro da tag p de interesse.
