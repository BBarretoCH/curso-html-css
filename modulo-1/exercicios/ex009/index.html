Listas Ordenadas

Quando criamos listas de algo onde a ordem importa chamamos isso de Ordered list (ol). Para criar uma ordered list devemos colocar a tag <ol> para delimitar a lista e a tag <li> para listar os itens.
li = list item

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5
OBS: A TAG ol é de fechamento obrigatório e a tag il de fechamento opcional segundo a W3C.
A tag ol possui um parâmetro type que determina o tipo de marcador da lista atual. São eles:
    1- Valor padrão. Cria listas numerada. Ex: 1.2.3.4.5
    A- Cria lista com ordenação alfabética maiúscula. Ex: A. B. C. D. E.
    a- Cria lista com ordenação alfabética minúscula. Ex: a. b. c. d. e.
    I- Cria lista com ordenação em número romanos maiúsculos. Ex: I. II. III. IV. V.
    i- Cria lista com ordenação em número romanos minúsculos. Ex: i. ii. iii. iv. v.

Além disso podemos inserir o parâmetro start para determinar o inicio.
EX: <ol type = "I" start = "5"> então deverar aparecer da seguinte forma: V. VI. VII. VIII, IX, X...

Listas Não Ordenadas

Podemos também criar listas que não necessitam de ordem. Possuem o termo Unordered list <ul>, são chamadas também como lista com marcadores. Podemos criar a lista da mesma forma que fizemos anteriormente, porém ao invés de utilizar a <ol>, utilizaremos a <ul>.

  • item 1
  • item 2
  • item 3
  • item 4
  • item 5
Como é possível ver, a estrutra é semelhante. Vale lembrar novamente que a tag ul tem fechamento obrigatório e a tag li tem fechamento opcional.
Assim como na ordered list, podemos inserir o parâmetro type para determinar o tipo de simbolo que irá aparecer na lista. Por padrão temos uma esfera preta totalmente preenchida, entretanto pode mudar utilizando as seguintes opções;
  • disc. Representa o padrão
  • circle. Representa um circulo sem preenchimento
  • square. Representa um quadrado preto preenchido (semelhante ao padrão)

Misturando Listas

É possível criar listas mistas colocando itens e subitens para criar algo parecido como uma ordenação em sumário.

  1. Item 1
  2. Item 2
    1. item 2.1
    2. item 2.2
    3. item 2.3
  3. Item 3
  4. Item 4
OBS: É possível misturar listas ordenadas (ol) com listas não ordenadas (ul)

Listas de definições

Esse tipo de lista não possui demarcações (ordenação ou simbolos de listas não ordenadas). Sua função é atribuir significado a um determinado termo.
Vejamos uns exemplos:

HTML
Linguagem de marcação criada para desenvolver conteúdo de sites
CSS
Linguagem de marcação para folhas de estilos em sites
JavaScript
Linguagem de programação usada para interatividade dentro de site

Para a montagem do código da lista acima devemos utilizar a tag <dl> para criar a lista de definição (exige fechamento). A Tag <dt>(EX: HTML) para especificar um termo e <dd>(Definição apresentada na lista) para explicar o que o termo significa. As tags <dt>e<dd> possuem fechamento opcional.
